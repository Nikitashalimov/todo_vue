# ToDoList Vue.js

#### Ссылка на проект

[ToDoList](https://nikitashalimov.github.io/todo_vue/)

#### Описание

ToDoList с адаптивной версткой и категориями (под разные задачи)

#### Технологии

Vue 3 + VueX 4

#### Инструкция 

###### Установка
```
npm install
```

###### Запуск проекта
```
npm run serve
```
